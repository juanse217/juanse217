

# 💫 About Me:
### Hi there, I'm Sebastian. You're all welcome to my world!

I’m a software engineering student expected to graduate in September 16th. I'm a Java lover, though I've used other technologies. At this point of my career, I'm focusing on creating secure and well structured REST APIs that can be a part of a microservices system. And, of course, currently looking to apply my knowledge at a company that gives me the opportunity.

- ### I'm focusing on:
    - Java with Spring Boot.
    - For my databases, I mainly use *PostgreSQL*, but I also know just enough of *MongoDB*.
    - An API without security, is a great vulnerability, that's why I use *Spring Security* with its friend JWT, though, if required, we can use simpler               authentication!
    - *DDD*, I'm currently learning and applying DDD on my projects because I want to start using a hexagonal architecture!
- ### I’m looking to collaborate on any project you think I can fit in! I love this, and I'd like to put my knowledge to good use.
- ### You can find my pinned projects:
    - *project_task_management*: An API to manage projects which contain tasks. Here you can have different users like project managers, users and the application admins.
    - *event_management*: allows clients to manage their events, participants and the registrations generated when a user registers to an event.
    - *movie_manager*: clients will handle movies with a simple CRUD using PostgreSQL as the database.
    - *product_catalog*: an API exposing simple and more complex endpoints to manage products using MongoDB as the database.

- ### What's next to learn:
    - Hexagonal architecture to bring a more escalable acrhitecture to my APIs.
    - System Design to be able to understand systems and escalate them understanding the trade-offs.
    - CI/CD to deploy the APIs.
    - Angular to have a frontend for my APIs and expose the services I have.
